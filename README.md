# Instagram Usage, Engagement & Ad Performance Dashboard

## Overview
This project analyzes Instagram user behavior, engagement metrics, and advertising performance using Power BI.

## Dataset
Instagram Usage & Lifestyle Dataset  
Data was cleaned and modeled using a star schema.

## Dashboard Highlights
- User demographics and usage trends
- Engagement analysis by age, gender, and country
- Ad reach and click effectiveness by country
